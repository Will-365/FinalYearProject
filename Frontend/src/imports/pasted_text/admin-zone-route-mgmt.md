Redesign only the content area for Zone & Route Management.

Screen Purpose

This screen allows administrators to:

Define and manage geographic service areas

Assign collectors to zones

Configure collection rules per zone

Monitor zone-level performance

This is a geographic and operational control interface.

Layout Structure

Use a 3-section layout:

LEFT: Zone Hierarchy Tree
CENTER: Zone Details & Configuration
RIGHT: Collector Assignment & Zone Performance

🔹 LEFT PANEL – Zone Hierarchy Tree

Display hierarchical structure:

District

Sector

Cell

Village

Features:

Expand / collapse nodes

Add new (District / Sector / Cell / Village)

Edit name

Delete (with confirmation)

Search zone

Each node clickable → loads details in center panel

🔹 CENTER PANEL – Zone Details & Configuration

When a zone is selected, display:

Basic Information

Zone Name

Level (District / Sector / Cell / Village)

Parent Zone

Status (Active / Inactive toggle)

Service Configuration

Collection frequency (e.g. 2x per week dropdown)

Allowed waste types:

Organic

Plastic

General

Priority level (High / Medium / Low)

Address Coverage Summary

Number of registered residents

Number of active collectors

Number of pending pickups

Total waste collected (monthly)

🔹 RIGHT PANEL – Collector Assignment

Display:

Assigned Collectors

Table:

Collector Name

Status (Active / Offline)

Current workload

Performance %

Buttons:

Assign Collector

Remove Collector

Assign Collector Modal

Dropdown of available collectors

Filter by availability

Show workload indicator

Confirm assignment

🔹 Zone Performance Snapshot (Below)

Mini analytics:

Total requests (this week)

Completed pickups

Missed pickups

Efficiency %

Use small charts or progress bars.

Smart System Logic

Include:

Warning if:

No collector assigned to zone

High number of missed pickups

Highlight overloaded zones

Prevent assigning same collector to conflicting zones
